## Installation

- `npm init`
- `npm install inquirer`
- `npm install mysql2`
- `npm install console-table`
- `npm install figlet`
- `npm install chalk`
- `npm install nodemon`
- `npm install validator`
  
## Usage
Use the Command Line after installing node dependencies to track and manage employee's within your company.
Use:

`npm start`
`node server.js`
